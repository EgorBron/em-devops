# em-devops

Тестовое задание на позицию Junior DevOps.

Построено на стеке из Python 3.14 и Docker.


## Сборка и запуск

1. Склонируйте репозиторий командой ниже:

  git clone https://github.com/EgorBron/em-devops

2. Соберите образ приложения перед запуском:

  docker build -f backend/Dockerfile -t my-app:1.0.0 backend/

3. Запустите приложение:

  docker run -p 8080:8080 my-app:1.0.0

4. Вы можете сменить адрес и порт, на котором слушает приложение, передав переменные окружения через параметр `-e`:

  docker run -p 9000:9000 -e BACKEND_PORT=9000 -e BACKEND_HOST=0.0.0.0 my-app:1.0.0

5. Проверьте работоспособность приложения с помощью `curl`:

  curl localhost:8080 -v


## Архитектура приложения

1. Запрос от клиента попадает на пробрасываемый порт (по умолчанию - 8080).
2. Docker направляет запрос в контейнер приложения.
3. Веб-сервер на базе модуля `http.server` возвращает строку `Hello from Effective Mobile!` в качестве ответа со статусным кодом 200.
4. Для конфигурации сервер использует переменные окружения `BACKEND_PORT` и `BACKEND_HOST`; последняя играет роль только при разработке вне контейнера.
